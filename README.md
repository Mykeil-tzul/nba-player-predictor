# 🏀 Luka Dončić Points Predictor
https://nba-player-predictor-bytzul.streamlit.app/ 

A Streamlit-powered NBA analytics app that predicts Luka Dončić's next game scoring performance using:

- 📊 Rolling averages (last 3 & 5 games)
- 📈 Advanced stats (AST, TRB, GmSc, shooting %)
- 🤖 Linear regression model
- 🎛 Interactive UI for custom predictions

---

## 🚀 Features

- Predict Luka's next game points
- Adjust game inputs (MP, FGA, 3PA, FTA)
- View recent scoring trends
- Compare actual vs predicted performance
- Analyze model accuracy (RMSE, R²)

---

## 🧠 Tech Stack

- Python
- Pandas
- Scikit-learn
- Streamlit
- Matplotlib

---
